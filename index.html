About Me

I'm a fifth year computer engineering student at Simon Fraser University with an interest in both hardware and software. My latest co-op at Intel had me working on firmware for embedded systems, which I found both interesting and rewarding especially since it required in depth knowledge of both the hardware and software used in the product. When I'm not doing my computer nerd stuff I enjoy being a pianist and hobby aquarist as well as hiking, biking, and skating.

Projects

Growing up in a small town meant a lot of my learning was, and still is, self-directed. Projects are a great way to hone and practice your skills. It started with small programs and games in highschool and moved onto much more complicated projects combining my knowledge of both software and hardware. Below are a few projects that I am particularly proud of.

LocalSonic - Indoor Navigation

For my capstone project I worked in a team of engineers to create a local guidance system based on ultrasonic distance calculation. The project can be found on Github.

Arduino Door Lock

Using an Arduino, an RFID sensor, and a servo I was able to implement a door lock that can be toggled from outside using a key card. The code can be found on my GitHub.

Home Automation

With EventGhost and Tasker I am able to automate my stereo and the power cycle of my computer. I hope to automate my lighting soon. Code can be found on Github.
