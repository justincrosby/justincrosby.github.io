Projects

Growing up in a small town meant a lot of my learning was, and still is, self-directed. Projects are a great way to hone and practice your skills. It started with small programs and games in highschool and moved onto much more complicated projects combining my knowledge of both software and hardware. Below are a couple projects that I am particularly proud of and make use of every day.

Arduino Door Lock

Using an Arduino, an RFID sensor, and a servo I was able to implement a door lock that can be toggled from outside using a key card. The code can be found on my GitHub.

Home Automation

With EventGhost and Tasker I am able to automate my stereo and the power cycle of my computer. I hope to automate my lighting soon. Code can be found on Github.
